# README

### DEPENDENCIES
* Required ruby version: 2.7.4

### Setup
* clone the repo
* cd /cultural_event_be
* bundle install
* rails db:migrate
* rails s

### API ENDPOINTS
* GET /events
  lists all events

* POST /create_gorki_events
   - creates events from gorki

* POST /create_co_berlin_events
   - creates events from co_berlin
